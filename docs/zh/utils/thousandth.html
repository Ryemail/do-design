# 数值千分位

# thousandth

对传入的数值或字符串数值进行千分位处理,第一位为参数数值或者数字字符串(必填)，第二位参数为千分位分隔符，默认逗号(选填)， 第三位参数为小数点分隔符，默认点(选填)，第四位参数为数值精确度，默认不计算(选填)

# 默认参数

thousandth(3000, ',', '.');

# 示例

thousandth('3000'); // 输出3,000

thousandth('4343434.89'); //输出 4,343,434.89
